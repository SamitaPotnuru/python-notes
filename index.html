1. What is Python basic

Definition: Python is a simple, high-level, interpreted programming language used for almost everything.

Example
print("Hello, World!")
# Output: Hello, World!

2. Variables basic

Definition: A variable is a name that stores a value in memory.

name = "Alex"
age = 20
print(name, age)
# Output: Alex 20

3. Data Types basic

Definition: Data types tell Python what kind of value a variable holds (number, text, etc.).

a = 10          # int
b = 3.14        # float
c = "hi"        # str
d = True        # bool
print(type(a), type(b), type(c), type(d))

4. Operators basic

Definition: Operators are symbols that perform operations on values like +, -, *, /, %, ==.

print(5 + 2)   # 7
print(5 // 2)  # 2  (floor div)
print(5 ** 2)  # 25 (power)
print(5 == 5)  # True

5. Input / Output basic

Definition: input() reads from the user, print() shows output on screen.

name = input("Your name: ")
print("Hello,", name)

6. Strings basic

Definition: A string is a sequence of characters written inside quotes.

s = "Python"
print(s.upper())     # PYTHON
print(s[0:3])        # Pyt
print(len(s))        # 6
print(f"Hi {s}!")    # Hi Python!

7. If / Else basic

Definition: Conditions let your program choose what to do based on true/false.

age = 18
if age >= 18:
    print("Adult")
elif age > 12:
    print("Teen")
else:
    print("Kid")

8. Loops basic

Definition: Loops repeat a block of code multiple times.

for i in range(3):
    print("Hi", i)

n = 0
while n < 3:
    print(n)
    n += 1

9. Lists basic

Definition: A list is an ordered, changeable collection of items written in [ ].

fruits = ["apple", "banana", "cherry"]
fruits.append("mango")
print(fruits[1])    # banana
print(len(fruits))  # 4

10. Tuples basic

Definition: A tuple is an ordered, unchangeable collection written in ( ).

point = (3, 5)
print(point[0])  # 3
# point[0] = 9  -> ❌ Error

11. Sets basic

Definition: A set is an unordered collection of unique items written in { }.

nums = {1, 2, 2, 3}
print(nums)        # {1, 2, 3}
nums.add(4)
print(nums)

12. Dictionaries basic

Definition: A dictionary stores data as key:value pairs in { }.

student = {"name": "Sam", "age": 20}
print(student["name"])     # Sam
student["grade"] = "A"
print(student)

13. Functions intermediate

Definition: A function is a reusable block of code that performs a task.

def greet(name):
    return f"Hello, {name}"

print(greet("Alex"))   # Hello, Alex

14. Lambda Functions intermediate

Definition: A lambda is a small one-line anonymous function.

square = lambda x: x * x
print(square(5))    # 25

15. List Comprehensions intermediate

Definition: A short way to create a new list from an existing iterable.

squares = [x*x for x in range(5)]
print(squares)   # [0, 1, 4, 9, 16]

evens = [x for x in range(10) if x % 2 == 0]
print(evens)     # [0, 2, 4, 6, 8]

16. Modules intermediate

Definition: A module is a Python file you can import to reuse its code.

import math
print(math.sqrt(16))    # 4.0
print(math.pi)          # 3.1415...

17. File Handling intermediate

Definition: File handling lets you read or write data to files on disk.

with open("notes.txt", "w") as f:
    f.write("Hello file!")

with open("notes.txt") as f:
    print(f.read())     # Hello file!

18. Exception Handling intermediate

Definition: A way to catch and handle errors so the program doesn't crash.

try:
    x = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero")
finally:
    print("Done")

19. OOP — Classes & Objects advanced

Definition: A class is a blueprint; an object is an instance created from that blueprint.

class Dog:
    def __init__(self, name):
        self.name = name
    def bark(self):
        print(self.name, "says woof!")

d = Dog("Rex")
d.bark()    # Rex says woof!

20. Inheritance advanced

Definition: Inheritance lets one class reuse the properties and methods of another.

class Animal:
    def speak(self): print("Some sound")

class Cat(Animal):
    def speak(self): print("Meow")

Cat().speak()   # Meow

21. Iterators advanced

Definition: An iterator is an object that returns items one by one using next().

nums = iter([1, 2, 3])
print(next(nums))   # 1
print(next(nums))   # 2

22. Generators advanced

Definition: A generator is a function that yields values lazily, one at a time.

def count_up(n):
    i = 1
    while i <= n:
        yield i
        i += 1

for x in count_up(3):
    print(x)   # 1 2 3

23. Decorators advanced

Definition: A decorator is a function that adds extra behavior to another function.

def loud(func):
    def wrapper():
        print("** START **")
        func()
        print("** END **")
    return wrapper

@loud
def hello():
    print("Hello!")

hello()

24. *args and **kwargs advanced

Definition: *args takes many positional arguments; **kwargs takes many keyword arguments.

def show(*args, **kwargs):
    print(args)      # tuple
    print(kwargs)    # dict

show(1, 2, 3, name="Alex", age=20)

25. Async / Await advanced

Definition: Async code lets the program do other work while waiting for slow tasks.

import asyncio

async def main():
    print("start")
    await asyncio.sleep(1)
    print("done")

asyncio.run(main())
